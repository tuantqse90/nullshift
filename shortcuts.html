02

Action Shortcuts

// quick actions without touching the mouse

Ctrl+K
Open the Command Palette. Search and execute any command with fuzzy matching. On Mac, use Cmd+K.
/
Focus Search. Opens the global search overlay or focuses the blog search input on the blog page.
?
Show Keyboard Shortcuts help overlay. Quick reference without leaving your current page.
Esc
Close all overlays. Dismisses modals, search, command palette, mobile nav, and agent panels.
03

Terminal Shortcuts

// the interactive terminal easter egg

Terminal Icon
Toggle the interactive terminal. Click the terminal icon (bottom-right on the home page) to open/close the embedded shell.
Enter
Execute command. Submits the current input line in the interactive terminal.
ArrowUp
Previous command. Scroll through command history in the interactive terminal.
ArrowUp/ArrowDown
Navigate command palette items. Use arrow keys to move selection up and down in the command list.
Enter
Execute selected command. Runs the currently highlighted item in the command palette.
04

Usage Tips

// maximize your efficiency

> Sequential Keys

Navigation shortcuts use a g prefix. Press g first, then within 800ms press the target letter. If you miss the window, just press g again.

> Input Fields

Keyboard shortcuts are automatically disabled when you are typing in input fields, textareas, or content-editable elements. No accidental navigation while composing text.

> Command Palette Power

The command palette (Ctrl+K) supports fuzzy matching. Type partial words like prod to quickly find "Go to Products". Use arrow keys to navigate, Enter to execute.

> Modifier Keys

Single-key shortcuts (/, ?, g) only fire without modifier keys held. If Ctrl, Alt, or Meta is pressed, the shortcut is ignored to avoid conflicts with browser shortcuts.

> Quick Reference

Forgot a shortcut? Press ? on any page to open the keyboard shortcuts help overlay. It works site-wide and closes with Esc.

> Accessibility

All shortcuts are supplementary. Every action available via keyboard is also accessible through click/tap navigation. Shortcuts enhance speed for power users but are never required.
