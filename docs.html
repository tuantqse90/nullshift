// API reference & integration guides

01. zkProof SDK

Generate and verify zero-knowledge proofs in your application. Supports Groth16 and Plonk proving systems.

Installation

npm install @nullshift/zk-sdk

Quick Start

import { Prover, Verifier } from '@nullshift/zk-sdk';

// Initialize prover with circuit
const prover = new Prover({
  circuit: './circuits/age-check.r1cs',
  provingKey: './keys/proving.key',
  system: 'groth16'
});

// Generate proof
const proof = await prover.prove({
  age: 25,
  threshold: 18
});

// Verify proof
const verifier = new Verifier({
  verificationKey: './keys/verification.key'
});

const isValid = await verifier.verify(proof);
console.log('Proof valid:', isValid); // true

API Reference

Prover.prove(inputs: object): Promise<Proof>

Generates a zero-knowledge proof for the given inputs against the loaded circuit.

Verifier.verify(proof: Proof): Promise<boolean>

Verifies a proof against the verification key. Returns true if valid.

Prover.exportCalldata(proof: Proof): string

Exports proof as Solidity calldata for on-chain verification.

02. AgentForge

Deploy autonomous AI agents with built-in risk management, monitoring, and privacy guardrails.

Installation

pip install agentforge

Create an Agent

from agentforge import Agent, Strategy, RiskManager

# Define trading strategy
class MomentumStrategy(Strategy):
    def evaluate(self, market_data):
        rsi = self.indicators.rsi(market_data, period=14)
        if rsi < 30:
            return self.signal('BUY', confidence=0.85)
        elif rsi > 70:
            return self.signal('SELL', confidence=0.80)
        return self.signal('HOLD')

# Create agent with risk limits
agent = Agent(
    name='momentum-scanner',
    strategy=MomentumStrategy(),
    risk=RiskManager(
        max_position_size=0.05,
        stop_loss=0.02,
        daily_loss_limit=0.10
    )
)

# Deploy agent
agent.start(
    data_feed='wss://feed.nullshift.sh/v1',
    interval='1m',
    mode='paper'  # 'paper' or 'live'
)

API Reference

Agent(name, strategy, risk): Agent

Creates a new autonomous agent with the specified strategy and risk parameters.

Agent.start(data_feed, interval, mode): void

Starts the agent. Mode can be 'paper' (simulated) or 'live' (real execution).

Agent.stop(): void

Gracefully stops the agent and closes all open positions.

Agent.status(): AgentStatus

Returns current agent status including P&L, open positions, and uptime.
